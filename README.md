# Dronly
🚁 Dronly - Backend Service API REST para logística de mensajería aérea autónoma. Gestiona flota y rutas con precisión geográfica.  Highlights: 
🔹 Motor geoespacial y Geofencing. 
🔹 Control de flota y autonomía. 
🔹 Optimización de rutas. 
🔹 Trazabilidad y estados en tiempo real.  
Arquitectura robusta para la logística de última milla aérea.
